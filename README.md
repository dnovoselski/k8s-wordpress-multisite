# 📝 WordPress Multisite on Kubernetes

Deploy a **WordPress Multisite** network on Kubernetes — a single installation serving multiple sites from one cluster. Includes MySQL backend, shared persistent storage, and subdomain/subdirectory routing via Ingress.

📖 **Full write-up:** [blog.twis7.com — Building a Multi-Blogging Platform with WordPress Multisite on Kubernetes](https://blog.twis7.com/index.php/2026/03/17/building-a-multi-blogging-platform-with-wordpress-multisite-on-kubernetes/)
